# my-project

> A Vue.js project

## Build setup

### [yarn](https://yarnpkg.com) or [npm](https://www.npmjs.com/)
``` bash
# Install dependencies
yarn install

# Server with hot reload at localhost:8080
yarn run start:dev

# Build for production with minification
yarn run build
```
